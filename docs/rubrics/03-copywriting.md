# コピーライティング & メッセージング（配点: 20点）

## 評価の目的
LPのテキストが「ターゲットに刺さり、行動を起こさせる」水準かを評価する。特徴の羅列ではなく顧客にとっての価値で語られているか、具体性があるか、読みやすいかを判定する。

---

## スコア段階定義

### 17-20点（優秀）
- ターゲットペルソナが明確に絞り込まれ、「自分のことだ」と感じさせる言葉選び
- **特徴ではなくベネフィット**で語られている（「AI搭載」→「対応時間を67%短縮」）
- 具体的な数字・事例・before/afterが含まれている
- 専門用語が適切にコントロールされている（ターゲットの語彙レベルに合致）
- 見出しだけ読んでもストーリーが理解できる（スキャンリーディング対応）
- 感情に訴えるフレーズと論理的な裏付けが両立している
- マイクロコピー（ボタン周り、フォーム周り）まで丁寧に設計されている

### 12-16点（合格）
- ターゲットは分かるが、ペルソナの解像度がやや低い
- ベネフィットはあるが具体性に欠ける（数字がない、「大幅に改善」等の曖昧表現）
- 専門用語がやや多いが、致命的ではない
- 見出しが内容を適切に要約しているが、訴求力は弱い
- マイクロコピーが汎用的（「送信」「お問い合わせ」等）

### 6-11点（改善必要）
- ターゲットが不明確で、誰に向けたLPか分からない
- 機能・スペックの羅列が中心で、ベネフィットへの変換がない
- 「業界最高水準」「圧倒的な」など根拠のない修飾語が多い
- テキスト量が多すぎて読む気が起きない
- 見出しが「機能紹介」「料金プラン」など説明的で訴求力ゼロ

### 0-5点（致命的）
- 文章が破綻している（誤字脱字、文法エラー多数）
- コピーがほぼ存在せず、画像だけのLP
- 内容が矛盾している、または誇大広告に該当するレベル
- ターゲットと言葉選びが完全にミスマッチ

## チェックリスト

- [ ] ヘッドラインに**ベネフィット**（結果・成果）が含まれているか
- [ ] 「特徴→ベネフィット」の変換がされているか（例：「24時間対応」→「深夜の問い合わせも逃さない」）
- [ ] **具体的な数字**が3箇所以上使われているか
- [ ] 1文が60文字以内に収まっているか（可読性）
- [ ] 箇条書き・太字・ハイライトで重要ポイントが強調されているか
- [ ] 見出しだけ読んで全体のストーリーが把握できるか
- [ ] 「あなた」「御社」など**読者に話しかける2人称**が使われているか
- [ ] 専門用語に注釈がある、または平易な言い換えがされているか
- [ ] 曖昧な修飾語（「圧倒的」「最高の」）に具体的根拠が伴っているか

---

## ベネフィット変換の公式

```
[機能] + 「だから」 + [ベネフィット]

例:
× AI自動応答機能を搭載
○ AI自動応答で、深夜の問い合わせにも即座に対応。対応漏れゼロを実現

× クラウド型で導入が簡単
○ 最短即日で導入完了。サーバー構築不要、ITスキル不問で誰でも使える
```

---

## Good/Bad パターン

### Good例
- 「月間3,000件の問い合わせを、2人のチームで回す方法」
  → 具体的数字 + ターゲットの課題にダイレクトに刺さる
- 各セクション見出し：「なぜ対応が追いつかないのか」→「自動化で解決した3社の事例」→「初期費用0円で始められる理由」
  → 見出しだけでストーリーが成立

### Bad例
- 「最先端テクノロジーで、ビジネスの可能性を広げます」
  → 何のテクノロジーか不明。ベネフィットが抽象的
- 見出し：「特徴」「料金」「よくある質問」「会社概要」
  → 訴求力ゼロの事務的な見出し

## 業界ベンチマーク
- ベネフィット型ヘッドラインは、特徴型に比べてCVRが平均28%高い
- 数字を含むヘッドラインは、含まないものに比べてクリック率が36%高い
- 2人称（あなた/御社）を使ったコピーは、3人称に比べてエンゲージメントが97%高い
