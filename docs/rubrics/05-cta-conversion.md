# CTA & コンバージョン設計（配点: 20点）

## 評価の目的
LP最大の目的である「コンバージョン」に直結する要素を評価する。CTAボタンの設計だけでなく、フォーム設計、心理的ハードルの低減、緊急性・限定性の演出など、行動を起こさせる仕組み全体を対象とする。

---

## スコア段階定義

### 17-20点（優秀）
- CTAの文言が**行動の価値を示す具体的な表現**（「無料で診断レポートを受け取る」等）
- CTAボタンが**3箇所以上**に配置（ファーストビュー、中間、最終セクション）
- ボタンのデザインが**ページ内で最も目立つ**要素になっている
- **心理的ハードルを下げる工夫**が複数ある（無料、簡単、リスクなし、所要時間の明示）
- **マイクロコピー**（ボタン下の補足テキスト）が効果的に使われている
- フォームのフィールド数が**必要最小限**（5項目以下）に絞られている
- **緊急性・限定性**の演出がある（期間限定、残り枠数、カウントダウン等）
- フォームにプログレスバー or ステップ表示がある（マルチステップの場合）

### 12-16点（合格）
- CTA文言にある程度の具体性がある（「資料をダウンロード」等）
- CTAは2箇所程度に配置
- ボタンは目立つがページ内の最優先要素ではない
- 心理的ハードルの低減は1つ程度（「無料」のみ等）
- フォームフィールド数は6〜8項目

### 6-11点（改善必要）
- CTA文言が汎用的（「送信」「お問い合わせ」「詳しくはこちら」）
- CTAが1箇所のみ、またはページ最下部にしかない
- ボタンが目立たない（周囲と同じ色、小さい）
- 心理的ハードルを下げる工夫がない
- フォームフィールドが多すぎる（9項目以上）
- 緊急性・限定性の要素がゼロ

### 0-5点（致命的）
- CTAが存在しない or 見つけられない
- CTAの遷移先が不明瞭（クリックしたら何が起こるか分からない）
- フォームが破綻している or 入力項目が15項目以上
- ユーザーに不安を与える設計（いきなり決済ページ、個人情報の大量要求）

## チェックリスト

### CTAボタン
- [ ] 文言が**価値訴求型**か（「送信」ではなく「無料で〇〇を手に入れる」）
- [ ] ボタンの色が**ページ内の他の要素と異なる**アクセントカラーか
- [ ] ボタンサイズが**十分に大きい**（モバイルでもタップしやすい）
- [ ] **3箇所以上**に配置されているか（ファーストビュー・中間・最終）
- [ ] ボタン直下に**マイクロコピー**があるか（「1分で完了」「営業電話なし」等）

### フォーム設計
- [ ] フィールド数が**5項目以下**か
- [ ] 各フィールドに**ラベル**と**プレースホルダー**があるか
- [ ] 必須/任意の区別が明確か
- [ ] エラーメッセージがリアルタイムで分かりやすいか
- [ ] プライバシーポリシーへの同意チェックがあるか

### 心理トリガー
- [ ] **無料・リスクなし**の表記があるか
- [ ] **所要時間**の明示があるか（「30秒で完了」等）
- [ ] **緊急性**の演出があるか（期間限定、残り枠数等）
- [ ] **限定性**の演出があるか（先着〇名、月間〇社限定等）
- [ ] **損失回避**のフレーミングがあるか（「今申し込まないと〇〇」）

## CTA文言の改善公式

```
× 汎用的な動詞
○ [無料/簡単] + [具体的な価値] + [を受け取る/を始める]

例:
× 「送信」「申し込み」「お問い合わせ」
○ 「無料で改善レポートを受け取る」
○ 「3分で見積もりを確認する」
○ 「今すぐ無料トライアルを始める」
```

## マイクロコピーの効果的パターン

| ボタン上 | ボタン下（マイクロコピー） |
|---------|------------------------|
| 無料で資料をダウンロード | クレジットカード不要 |
| 今すぐ相談する | 平均対応時間：2時間以内 |
| 無料診断を受ける | 3分で完了・営業電話なし |
| 導入事例集を見る | PDF形式・全24ページ |

---

## Good/Bad パターン

### Good例
- オレンジ色の大きなボタン「無料で改善プランを受け取る」+ ボタン下に「30秒で完了 | 営業電話は一切しません」
  → 価値訴求+ハードル低減+リスクリバーサルが揃っている
- フォーム：名前・メール・会社名の3項目のみ + プログレスバー「ステップ1/2」
  → 最小限のフィールドで心理的負担が小さい

### Bad例
- 灰色の小さなボタン「送信」（マイクロコピーなし）
  → 何が起こるか分からない。押す動機がない
- フォーム：氏名・フリガナ・メール・電話・会社名・部署・役職・住所・年齢・予算・導入時期・きっかけ（12項目）
  → 心理的負担が大きすぎて離脱必至

## 業界ベンチマーク
- フォームフィールドを5つ以下にすると、コンバージョンが120%向上

- マイクロコピーの追加だけでCVRが17%改善した事例あり
- 緊急性の演出（カウントダウン等）でCVRが平均9%改善
- CTA色をアクセントカラーに変更するだけでクリック率が21%向上
