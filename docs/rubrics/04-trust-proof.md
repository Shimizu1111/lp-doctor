# 信頼性 & 社会的証明（配点: 10点）

## 評価の目的
訪問者の「本当に大丈夫？」という不安を解消し、行動への心理的障壁を下げる要素が十分に配置されているかを評価する。信頼性は単独で機能するのではなく、コピーやCTAと連携して初めて効果を発揮する。

---

## スコア段階定義

### 9-10点（優秀）
- **お客様の声**が実名・顔写真・所属付きで3件以上掲載されている
- 導入実績が具体的な数字で示されている（「3,200社導入」「満足度98.5%」等）
- メディア掲載ロゴ、受賞歴、認定資格などの**権威性の裏付け**がある
- 具体的な成果を示すケーススタディ（before/after）がある
- 運営会社の情報が明確に掲載されている
- 返金保証・無料トライアルなど**リスクリバーサル**が提示されている

### 6-8点（合格）
- お客様の声はあるが、匿名または抽象的（「A社 ご担当者様」）
- 実績数字はあるが、1種類のみ
- 権威性の裏付けは弱い（自社調べのみ等）
- 運営会社情報はフッターにある程度
- リスクリバーサルが不十分

### 3-5点（改善必要）
- お客様の声がない、または明らかに創作っぽい
- 実績を示す数字がまったくない
- 権威性の裏付けがゼロ
- 運営者情報が不明確
- 第三者視点の証拠が皆無

### 0-2点（致命的）
- 信頼性に関する要素が一切ない
- 記載内容に矛盾や虚偽の疑いがある
- 連絡先・運営者情報が完全に不明
- むしろ不信感を抱かせる要素がある（怪しい煽り文句、非現実的な数字）

## チェックリスト

- [ ] **お客様の声/推薦文**が掲載されているか
- [ ] お客様の声に**実名・顔写真・所属**があるか（信頼度が段違い）
- [ ] **導入実績の数字**（社数、利用者数、満足度等）が明示されているか
- [ ] **メディア掲載・受賞歴・認定**などの第三者評価があるか
- [ ] **具体的な成果事例**（before/after、数字付き）があるか
- [ ] **運営会社情報**（会社名、所在地、代表者）が記載されているか
- [ ] **保証・返金・無料トライアル**などのリスクリバーサルがあるか
- [ ] ロゴウォール（取引先・メディアのロゴ一覧）があるか

## 信頼性要素の効果ランキング

| 順位 | 要素 | CVR改善効果 |
|------|------|-------------|
| 1 | 具体的な成果事例（数字付きbefore/after） | +34% |
| 2 | 実名・顔写真付きのお客様の声 | +25% |
| 3 | リスクリバーサル（返金保証等） | +21% |
| 4 | ロゴウォール（有名企業のロゴ） | +18% |
| 5 | メディア掲載・受賞歴 | +15% |
| 6 | 匿名のお客様の声 | +5% |

## Good/Bad パターン

### Good例
- 「株式会社〇〇 マーケティング部 田中太郎様」+ 顔写真 + 「導入3ヶ月で問い合わせ対応時間が72%削減。チームの残業もほぼゼロに」
  → 実名+具体的成果で信頼性が高い
- 「導入企業3,200社以上」の下にロゴウォール（SONY, Toyota等）
  → 数字+有名企業の権威で一気に信頼感UP

### Bad例
- 「利用者の声：とても良いサービスです！（30代 女性）」
  → 抽象的で創作の疑い。具体性ゼロ
- 「顧客満足度No.1」（調査元・条件の記載なし）
  → 根拠不明で逆に不信感を生む
