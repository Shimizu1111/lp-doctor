# ビジュアルデザイン & UX（配点: 15点）

## 評価の目的
LPのビジュアル面が「読みやすさ」「情報の優先順位」「ブランドの信頼感」を適切に表現しているかを評価する。デザインは装飾ではなく、コンバージョンを最大化するための機能として評価する。

---

## スコア段階定義

### 13-15点（優秀）
- 視覚的ヒエラルキーが明確（見出し→小見出し→本文→補足の階層）
- **余白（ホワイトスペース）**が十分に取られ、情報が呼吸している
- CTAボタンがページ内で**最も目立つ色**で配色されている
- 一貫したカラーパレット（3色以内+アクセント1色）が使われている
- フォントの種類が2種類以内に統一されている
- 画像・アイコンが統一されたトーンで、訴求内容を補強している
- レスポンシブ対応が考慮されたレイアウト（1カラム、タップ可能なサイズ）
- Z型 or F型の視線誘導に沿ったレイアウト

### 9-12点（合格）
- 基本的な階層構造はあるが、一部メリハリが弱い
- 余白はあるが詰まっている箇所がある
- CTAは目立つがページ内最優先ではない
- カラーは統一感があるが、アクセントが弱い
- 画像のクオリティにばらつきがある
- モバイル対応は基本的にできていそう

### 5-8点（改善必要）
- 視覚的ヒエラルキーが不明確（何を最初に読むべきか分からない）
- 余白不足で窮屈な印象
- CTAと他の要素の区別がつきにくい
- 4色以上が使われ、統一感がない
- ストックフォトの過度な使用、画質のばらつき
- テキストが画像の上に読みにくく配置されている

### 0-4点（致命的）
- レイアウトが崩壊している
- フォントサイズが小さすぎて読めない（16px未満の本文）
- 色のコントラスト比が極端に低い（アクセシビリティ違反）
- 画像が低解像度、または表示されていない
- 全体的に素人感が強く、ブランドの信頼性を毀損している

## チェックリスト

### タイポグラフィ
- [ ] 本文フォントサイズが**16px以上**か
- [ ] 行間（line-height）が**1.5〜1.8**の範囲か
- [ ] 1行の文字数が**35〜45文字**以内か（全角）
- [ ] 見出しと本文のサイズ比が**1.5倍以上**あるか
- [ ] フォントの種類が**2種類以内**に統一されているか

### カラー
- [ ] **メインカラー**（ブランド色）、**サブカラー**、**アクセントカラー**の3色構成か
- [ ] CTAボタンに**アクセントカラー**（他で使っていない色）が使われているか
- [ ] 背景色と文字色のコントラスト比が**4.5:1以上**あるか
- [ ] 色だけに依存せず、形状や文字でも情報が伝わるか

### レイアウト
- [ ] セクション間に**十分な余白**（40px以上）があるか
- [ ] 要素のグルーピング（近接の法則）が適切か
- [ ] **F型 or Z型**の自然な視線誘導に沿っているか
- [ ] 情報量が多い箇所はアコーディオンやタブで整理されているか

### 画像・ビジュアル
- [ ] 画像が**高解像度**で統一されたトーンか
- [ ] アイコンのスタイルが**統一**されているか（線画なら全て線画）
- [ ] イラスト/写真が**訴求内容を補強**しているか（装飾だけになっていないか）

## 視線誘導パターン

### F型パターン（テキスト中心のLP向け）
```
███████████████████
███████████████████
███████████
███████████
███████
███████
████
```
→ 見出し・冒頭に重要情報を配置。左揃えが基本。

### Z型パターン（ビジュアル中心のLP向け）
```
1 ─────────── 2
      ╲
        ╲
3 ─────────── 4
```
→ 左上にロゴ、右上にCTA、左下に証拠、右下にCTA。

---

## Good/Bad パターン

### Good例
- 白背景 + 黒テキスト + 青アクセント（CTA）のシンプルな3色構成
  → 視線がCTAに自然に集中する
- セクションごとに背景色を白/薄灰で交互に切り替え
  → 情報の区切りが視覚的に明確

### Bad例
- 赤、青、黄、緑、紫が混在するカラフルなデザイン
  → 視線が分散し、どこが重要か分からない
- 本文14px + 行間1.2 + 1行60文字
  → 可読性が低く、読む気が失せる
- 背景に全面写真、その上に白テキスト（コントラスト不足）
  → テキストが読めない箇所が発生

## 業界ベンチマーク
- 余白を適切に確保したLPは、詰め込みLPに比べて理解度が20%向上
- CTA色をアクセントカラーに変更するだけでクリック率が21%向上
- モバイルファーストデザインのLPは、デスクトップ優先LPに比べてモバイルCVRが32.5%高い
- 画像の高品質化だけでCVRが40%改善した事例あり
